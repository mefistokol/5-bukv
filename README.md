Игра 5 букв — Угадай слово!

Русскоязычный аналог игры **Wordle**, написанный на Python с использованием tkinter. Угадайте загаданное слово из 5 букв за 6 попыток!

Тёмная тема оформления с фирменной жёлтой цветовой схемой в стиле **Т-Банка**.


---

## 🎮 Правила игры

1. Компьютер загадывает случайное русское слово из **5 букв**.
2. У вас есть **6 попыток**, чтобы его угадать.
3. После каждой попытки буквы подсвечиваются цветом:

| Цвет | Значение |
|------|----------|
| 🟨 **Жёлтый** | Буква стоит на **правильной** позиции |
| ⬜ **Белый** | Буква **есть** в слове, но на другой позиции |
| ⬛ **Серый** | Буквы **нет** в загаданном слове |

4. Вводимое слово должно существовать в словаре (`words_5.txt`).
5. Буква **«ё»** автоматически заменяется на **«е»**.
---

## 🖥️ Интерфейс

- **Экранная клавиатура** — раскладка ЙЦУКЕН с цветовой индикацией использованных букв
- **Физическая клавиатура** — поддержка ввода русских букв, Enter и Backspace
- **Анимация ошибки** — «тряска» поля ввода с красной рамкой при вводе несуществующего слова
- **Всплывающие сообщения** — подсказки при ошибках:
  - *«Нужно ровно 5 букв»* — если введено недостаточно букв
  - *«Нет такого слова в словаре»* — если слово отсутствует в словаре
- **Счётчик попыток** — отображение номера текущей попытки
- **Экран результата** — оверлей с итогами игры (победа/поражение)
- **Кнопка «Играть ещё»** — для начала новой партии

## 🚀 Запуск

### Требования

- **Python 3.x** (с модулем `tkinter`)
- Файл словаря **`words_5.txt`** в одной директории с игрой

> 💡 Используются только модули стандартной библиотеки Python: `tkinter`, `random`, `os`, `sys` — установка дополнительных пакетов не требуется.

### Как запустить

```bash
python Bukv5.py
```

Убедитесь, что файл `words_5.txt` (словарь слов, кодировка UTF-8, одно слово на строку) находится в той же папке, что и `Bukv5.py`.
